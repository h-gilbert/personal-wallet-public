Know your weekly spending power — income minus bills and budgets, calculated automatically. Transfer plans tell you exactly how much to move to each account. Track spending against each category in real time.

Set targets for what matters — a house deposit, a trip, an emergency fund. Link goals to accounts, set deadlines, and see exactly how much you need to save each week to get there.
