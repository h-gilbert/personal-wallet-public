# Personal Wallet - Showcase

A static showcase site for [Personal Wallet](https://github.com/h-gilbert/personal-wallet-backend), a self-hosted personal finance application.

## Live Site

[wallet.hamishgilbert.com](https://wallet.hamishgilbert.com)

## About

This site showcases the features of Personal Wallet with screenshots from both the web app and native iOS app. It is a static HTML/CSS page with no build step.

## Development

```bash
# Preview locally
python3 -m http.server 8000
```

## Deployment

Auto-deploys via GitHub Actions on push to `main`.
